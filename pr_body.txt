## 📝 Details
This PR restructures the core navigation experience by prioritizing the collaborative Community tab as the new application home screen. It also resolves a pre-existing compilation bottleneck in unit testing.

### 🚀 Key Features & Improvements
* **🌐 Community-First Navigation:** Re-architected `HomeScreen` and navigation components to set the `CommunityScreen` as the starting destination, moving the Shopping List tab to the second position.
* **📱 Unified Back Navigation:** Updated the Compose `BackHandler` logic to trigger the exit dialog natively from the new home screen (Community).
* **🧪 Test Suite Stability:** Resolved pre-existing compilation errors in `ProfileViewModelTest.kt` caused by constructor and model signature mismatches.

## 🧪 How Has This Been Tested?
* Compiled cleanly via dev unit tests: `.\gradlew.bat :app:compileDevDebugUnitTestKotlin`. All 24 tasks executed and compiled successfully.
